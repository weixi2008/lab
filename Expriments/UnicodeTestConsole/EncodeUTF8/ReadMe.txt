========================================================================
    EncodeUTF8 Project Overview
========================================================================

将所有的源文件，转换为UTF8 BOM编码。因为仅UTF8 BOM支持github的git diff比较。

	1. Windows平台版本，直接循环所有的子目录转换。
	2. 无需配置文件，转换所有的.h, .hpp, .c及.cpp。 不转换resource.h。
	3. 带BOM的UTF16 转为 UTF8，
	4. 无BOM头，认为是ANSI。转为UTF8。 
	5. 带BOM的UTF8， 不变。
	6. 回车换行，暂时不动。
